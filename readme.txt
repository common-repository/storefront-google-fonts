**How to use:**

1. Go to Appearance -> Customize
2. Click on Advance Typography
3. Make your Changes.


== Installation ==

#### Installing the plugin
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for **Storefront Google Fonts** and click "*Install now*"
1. Alternatively, download the plugin and upload the contents of `storefront-google-fonts.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
1. Activate the plugin
